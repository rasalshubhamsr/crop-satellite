CROP CLASSIFICATION USING SATELLITE IMAGES

INTRODUCTION

Agriculture is the backbone of many economies, including India. The project "Crop Classification Using Satellite Images" focuses on using remote sensing and machine learning to automatically identify crop types. It enables accurate, large-scale crop monitoring, reduces manual surveys, and supports better agricultural planning and decision-making.

PROBLEM STATEMENT

Farmers, researchers, and government bodies lack an automated, scalable, and cost-efficient system to classify and monitor different crop types over large areas using satellite imagery. Without timely and accurate crop data, planning and resource distribution become inefficient.

SOLUTION

This project aims to solve this by using satellite imagery and machine learning to classify crops over large geographical areas automatically.

OBJECTIVES

  • To develop an automated system to classify crop types using satellite images.
  • To Enable real-time, large-scale agricultural monitoring.
  • To Support government in subsidies, and planning.

TECHNOLOGY STACK

Satellite Data: Sentinel-2, Sentinel-1, PlanetScope
Programming: Python, Google Earth Engine
Machine Learning: Random Forest, SVM, CNN, LSTM
Tools: QGIS, SNAP, scikit-learn, TensorFlow, Keras
